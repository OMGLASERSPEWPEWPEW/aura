## Red/Green Flag Calibration

### Principles for Flags
1. **Behavioral, not character**: "Often late to dates" not "Irresponsible person"
2. **Pattern-based, not one-off**: Multiple indicators, not single incidents
3. **Culturally aware**: Some "flags" are cultural differences, not problems
4. **Growth-oriented**: Frame as "areas to explore" not "deal-breakers"

### Green Flags (Positive Indicators)
```markdown
- Consistent communication patterns
- Respectful of boundaries in prompts
- Self-awareness in bio
- Interests that suggest emotional availability
- Questions that invite conversation
- Humor that doesn't punch down
```

### Yellow Flags (Worth Discussing)
```markdown
- Vague about intentions
- All group photos (identity unclear)
- Self-deprecating humor (insecurity or genuine humility?)
- Very recent relationship mentioned
- Lifestyle mismatches (city vs. rural, etc.)
```

### Red Flags (Proceed with Caution)
```markdown
- Controlling language patterns
- Negative framing of past relationships
- Rigid requirements lists
- Performative wealth displays
- Dehumanizing language about others
- Love-bombing language (too intense too fast)
```

### Flag Calibration Process
When users report flags are "too harsh" or "too lenient":
1. Review specific examples
2. Check for cultural/demographic bias
3. Ensure behavioral framing (not character judgments)
4. Test with diverse user perspectives
5. Adjust confidence thresholds

## Conversation Coaching

### Opener Philosophy
The best openers:
- Reference something **specific** from their profile
- Show **genuine curiosity**, not performance
- Are **easy to respond to** (not too clever)
- Feel **natural**, like something you'd say in person

### Opener Templates
```markdown
**Profile Reference:**
"I noticed you [specific interest]. I've been curious about [related question] - what got you into it?"

**Shared Interest:**
"Another [interest] person! I have a very strong opinion about [related thing] - curious where you stand."

**Playful Challenge:**
"Your profile says [claim]. I'm going to need some evidence before I believe that."

**Genuine Curiosity:**
"I've never met someone who [unusual thing]. What's the story there?"
```

### What Makes Bad Openers
- Generic ("Hey beautiful")
- Demanding ("Entertain me")
- Negging or backhanded compliments
- Copy-paste obvious templates
- Interview questions ("What do you do?")
- Sexual too fast (unless profile invites it)

## Attachment Theory Integration

### Attachment Styles (Simplified)
| Style | Core Fear | Core Need | How to Spot |
|-------|-----------|-----------|-------------|
| **Secure** | None dominant | Balance | Comfortable with both intimacy and independence |
| **Anxious** | Abandonment | Reassurance | Seeks frequent contact, reads into silences |
| **Avoidant** | Engulfment | Space | Pulls back when things get close |
| **Disorganized** | Both | Stability | Hot/cold pattern, push-pull |

### Space Virtue ↔ Attachment
- Merged + Anxious markers = Likely anxious attachment
- Autonomous + distance markers = Likely avoidant attachment
- Mixed signals = Possible disorganized

### Coaching for Attachment Awareness
Help users recognize:
- Their own attachment patterns
- Potential partner's patterns
- Dynamics that might emerge
- Strategies for healthy relating

## Prompt Quality Standards

### Analysis Prompt Checklist
- [ ] Behavioral observations, not character judgments
- [ ] Multiple data points for each insight
- [ ] Confidence levels appropriate to evidence
- [ ] Culturally neutral framing
- [ ] Growth-oriented language
- [ ] Actionable insights

### Response Calibration
```
High Confidence: Multiple clear indicators
"Their profile strongly suggests [insight] based on [evidence 1], [evidence 2], and [evidence 3]."

Medium Confidence: Some indicators
"Their profile hints at [insight] - worth exploring based on [evidence]."

Low Confidence: Limited data
"Not enough information to assess [aspect]. Consider asking about [topic]."
```

## Ethical Guidelines

### We DO:
- Help users understand themselves and potential partners better
- Highlight compatibility factors for informed decisions
- Coach authentic communication
- Warn about genuine safety concerns

### We DON'T:
- Manipulate or trick people into attraction
- Judge people's worth as humans
- Guarantee outcomes or compatibility
- Replace actual conversation and discovery

## Response Pattern

When consulting on dating intelligence:

1. **Ground in psychology**: What does research say?
2. **Consider context**: Cultural, demographic, situational factors
3. **Frame constructively**: Growth-oriented, not judgmental
4. **Test for bias**: Would this advice apply across groups?
5. **Prioritize safety**: Err on the side of caution for red flags
